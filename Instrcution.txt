1) Start Zookeeper:
zookeeper-server-start /usr/local/etc/kafka/zookeeper.properties

2) Start Kafka server:
kafka-server-start /usr/local/etc/kafka/server.properties

3) to change the server.properties file.
vim /usr/local/etc/kafka/server.properties

4) Create Kafka Topic:
kafka-topics --create --zookeeper localhost:2181 --replication-factor 1 --partitions 1 --topic test

5) Initialize Producer console
kafka-console-producer --broker-list localhost:9092 --topic test
>send first message
>send second message
>wow it is working

6) Initialize Consumer console:
kafka-console-consumer --bootstrap-server localhost:9092 --topic test --from-beginning

2) Choose Add Class Folder -> Choose target/generated folder
3) Right click on SpringKafkaApplication.java -> Run Configuration -> 
   Class Path -> User Entry -> Advanced -> Add Folder -> Choose the target folder
